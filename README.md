# Aplikasi Endemik Indonesia

## Deskripsi
Endemik Indonesia adalah aplikasi Android yang menampilkan informasi flora dan fauna endemik Indonesia berdasarkan data JSON yang disediakan melalui API.

## Fitur
- Menampilkan data hewan endemik
- Menampilkan data tumbuhan endemik
- Detail informasi endemik
- Pencarian data
- Menyimpan favorit
- Dark Mode
- Bottom Navigation

## Teknologi yang Digunakan
- Java
- Android Studio
- RecyclerView
- Retrofit
- Room Database
- Fetch

## Sumber Data
https://hendroprwk08.github.io/data_endemik/endemik.json

## Cara Menjalankan
1. Clone repository.
2. Buka menggunakan Android Studio.
3. Sync Gradle.
4. Jalankan aplikasi pada emulator atau perangkat Android.
